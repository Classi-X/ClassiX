Add Students
Instructions:
You can add students either manually one by one, or upload a CSV file to add many students at once.

üìå Manual Entry:

  • Click the "Add Student" button to add student fields.
  • Fill in all the required details for each student:
    • Roll Number ‚Äì must be unique (not used before)
    • Full Name (First + Middle + Last) - Username ‚Äì must be unique
    • Email ‚Äì must be a valid email and not used for another student
    • Class ‚Äì select the class of the student
    • Stream ‚Äì only required if class is 11 or 12

üìÇ Upload via CSV File:

  • Click the "Choose File" button below to upload a CSV file.
  • Use the exact format as shown in the sample file. Each row must contain:
    • Roll Number
    • Full Name (First + Middle + Last) - Username
    • Email
    • Class
    • Stream (leave blank if class is not 11 or 12)
  • Do not change the column names in the first row.
  • Make sure all rows are complete and follow the same order.
  • If there's any error in a row (duplicate email, missing class, etc.), that student won't be added.

üîê How Students Will Login:

  • Passwords are not created here.
  • After registration, students can go to the login page and click "Forgot Password".
  • They will enter their email (that you provided here) and receive an OTP.
  • After entering the OTP, they can create their own password.
